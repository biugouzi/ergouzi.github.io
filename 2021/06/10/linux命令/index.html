##文件操作

###增：

touch [文件名.后缀]  //新建文件
vi: wq //编辑保存并退出
echo > [名称].txt //写入内容至一个文本文件 若目录无改名字的文本文件 则会自动创建
mkdir -p //新建一个文件夹 -p：递归创建

###删:

rmdir //只能删除空文件夹
rm //删除一个文件/文件夹
-r //将目录及以下文件全都删除
-f //无需确认 直接删除
-i //删除是会提示 默认为提示
rm -rf * //删除改目录下所有文件 谨慎使用
fg %[工作号] //结束某个在后台的任务

###查：

ls //查看当前目录
-a //查看当前目录所有文件 包括隐藏文件
-l //查看当前目录文件权限
-t //将文件依建立时间之先后次序列出
cat //查看文件内容
tac //倒序查看文件内容
head -n //显示头几行文本内容 默认为10行
tail -n //显示末尾几行文本内容 默认为10行
tail -f //实时显示新增文本内容
grep //查找相关字符串内容
find //索引搜索
ps -aux //查询所有进程
whereis * //查看*在什么路径
jobs [选项] //查看后台
more [选项] [文件名] //显示文件第一页
whoami //查看用户
w=who //查看所有登录的用户

###改：

cp // 复制一个文件/文件夹
-r //若给出的源文件是个目录文件，将复制改目录下所有的子目录与文件
-p //复制文件，并会会把修改时间和访问权限也复制到新文件中
-f //覆盖已存在的目标文件不会提示
cat /etc/passwd > ./passwd 修改文件内容至passwd
mv [文件名] [目标目录] //移动文件至目标目录
mv [文件名] [新文件名] //修改文件名
touch -t [时间] [文件名] 修改文件创建世界/最后修改时间

##查看系统信息

uname 命令：
uname --help //查看帮助
uname -n //显示计算机名
uname -s //显示系统名称
uname -a //显示系统信息
uname -m //显示计算机类型
uanme -v //显示系统版本与时间
uname -r //显示系统发行编号

##Linux按键命令

Tab //自动补全命令
Ctrl+l //清屏
Ctrl+u //清空至首行
Ctrl+c //取消本次命令执行
Ctrl+z //将程序放入后台

##Linux压缩命令

tar [选项] 源文件或目录.tar.gz // 打包文件
tar 打包与压缩:
tar cvf a.tar ./ //打包本地文件名称为a.tar
tar xvf a.tar //解压a.tar文件

tar zcvf b.tar.gz ./* //打包所有文件为b.tar.gz
tar zxvf b.tar.gz  //解压文件 b.tar.gz

##PIng命令

ping [目标主机/域名] //对目标主机发送数据包，网络测试
traceroute baidu.com //自由追踪

##Linux命令补充：

关机指令： 
poweroff //立刻关机
halt //立刻关机
init命令：  
--run level 0: 关机
--run level 3: 纯文本模式
--run level 6: 重新启动
reboot //立即重启
Shutdown [选项] 时间
选项： 
-c: 取消前一个关机命令
-h: 关机
-r：重启
用户登录命令; 
login //登录
基本语法：login [name][-p][-h主机名称]
用户退出命令： 
exit //退出控制台
logout //注销用户
apt-get:  
sudo apt-get update 更新软件列表
sudo apt-get upgrade 更新软件
sudo apt-get install banner 安装一个文字图形输出
