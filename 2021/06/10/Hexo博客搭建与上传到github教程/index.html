#Hexo博客搭建

##第一步：安装choco工具

进入choco官网：https://chocolatey.org/
以管理员身份运行终端
复制安装命令：
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]: : SecurityProtocol = [System.Net.ServicePointManager]: : SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString(‘https://chocolatey.org/install.ps1'))

检查是否安装成功 命令行输入choco

choco

使用choco 安装Git

choco install git

输入git查看是否安装成功

git

##第二步：安装Hexo

打开Hexo官网:https://hexo.io
复制安装代码

npm install hexo-cli -g //使用apm安装Hexo

cd D: [可自己选择盘，此处为演示步骤] //进入D盘

hexo init biuergouzi[可自己定义名称，此处为演示步骤] //在目标目录盘，创建一个名为biuergouzi的文件夹

cd biuergouzi //进入创建的文件夹

npm install 使用npm安装所需文件

hexo s（hexo server命令） 查看

浏览器中输入http://localhost:4000 查看网页

##第三步：上传至github

1.创建一个github账户
2.在github创建一个仓库，仓库名用自己的github名称命名 例如:Biuergouzi.github.io
3.在终端中输入-配置本地账户：

git config –global user.name “github的用户名”
git config –global user.email “github的邮箱”

ssh-keygen -t rsa -C “上面输入的邮箱” //生成密钥SSH key

cat ~/.ssh/id_rsa.pub //查看SSH key (记录这个key)

4.打开github 点击头像右侧的倒三角 点击settings 设置
5.点击左侧栏中的 SSH and GPG keys — 然后点击SSH keys 右侧的新建SSH key //新建SSH key

Title： 输入一个名称
key：输入之前记录的key
点击add SSH key //添加SSH key

6.终端中输入–安装部署插件：

npm install hexo-deployer-git –save //安装一个部署插件

7.修改博客的ergouzi文件夹中配置文件
在最底下 deploy:
type: git
repository: https://github.com/[github用户名]/[创建的仓库名字].git
branch: master

8.终端中输入:

hexo g //重新配置ergouzi文件夹中的文件
hexo d //将本地的文件不输入到github仓库

##第四步：浏览器打开查看博客
浏览器中打开链接：[github用户名].github.io

例如：biuergouzi.github.io

##完成！
