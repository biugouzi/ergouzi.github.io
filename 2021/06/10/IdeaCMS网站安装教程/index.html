Windows server 2008虚拟机中IdeaCMS网站安装教程

六月 10, 2021
  • 基础

IdeaCMS网站安装教程：

<!-以下步骤皆为在虚拟机中操作–！> 此教程为已在windows server中安装iis服务

第一步：下载ideaCMS源码包 导入至Windows server 2008中

第二步解压至C盘目录ideaCMS文件夹中

第三步：打开服务器管理器

第四步：建立网站

右击网站—点击添加网站—网站名称随意—路径选择之前解压的文件夹

物理路径选择之前解压在C盘的ideacms

最后点击确定按钮

第五步：配置

点击ideacms网站—找到ASP配置—找到用父路径将false==>true

找到默认文档配置—添加默认文档—名称填写index.asp

打开应用程序池—找到ideacms—右击选择高级设置—修改启用32位应用程序为True

在c盘\ideacms\inc文件夹中找到config.asp配置文件—记事本打开编辑

—修改databaseType=“asp”==>databaseType=”sql”—保存并退出文件

第六步：连接数据库

打开SQL server Management Studio

— 右击数据库—选择新建数据库

—数据库名称填写ideacms

–右击ideacms—任务—导入数据

—数据源选择Mincrosoft Access—文件名浏览c:\ideacm\data*.mdb 文件

然后点击下一步

—到表和试图的时候 将所有表都选择

—默认下一步—导入成功

第七步：浏览器打开

在浏览器中输入框中输入虚拟机ip即可打开网页
