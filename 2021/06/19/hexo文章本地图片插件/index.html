hexo文章本地图片插件

六月 19, 2021
  • 基础

一、安装插件与修改配置文件

npm install https://github.com/7ym0n/hexo-asset-image –sa   //修改过的插件，原本的插件有Buf，这个已经经过修改 安装后就无需修改了。

打开hexo博客根目录的配置文件’_config.yml’ 修改为以下内容

post_asset_folder: false ==> true

二、创建文章

进入hexo博客文章目录中，在当前目录打开终端/cmd

hexo new “文章名”    //创建一个文章

创建完成后，会在文章目录中创建一个与文章名相同的文件夹

这个文件夹就是用来放图片的文件夹

三、修改Typora偏好设置

打开Typora，点击左上角文件—点击偏好设置

在偏好设置中点击–图像

将无特殊操作修改为复制到指定路径，然后将【对本地位置的图片应用上述规则】以及【优先使用相对路径勾上】

四、插入图片/截图

将图片放入文件夹中，在文章内容中输入

第二种放入截图的方式就十分简单了

使用截图工具 QQ或者微信的都可以，截图后直接复制到文章内，会自动添加图片到文件夹中

这样就可以正常使用了~
